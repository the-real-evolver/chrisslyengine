Building Chrissly for Android
========================================================================
- install Android Studio Giraffe | 2022.3.1 or higher
- open project 'chrisslyengine/android' in Android Studio
- press press 'CTRL + F9' to build the application
- run 'exportassets.cmd'

Running the sample with Android Studio
========================================================================
- press 'SHIFT + F10' in Android Studio to start the sample application
